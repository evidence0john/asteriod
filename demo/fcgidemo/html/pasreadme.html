你将要使用的 demo 的名字吧，我还没有想好,所以我简单的说说这是怎么用的吧 -_-#

你可以像发帖或者发朋友圈一样发出一个主题，试试看发一个新的主题，可以用来测试，也可以留言给我，目前不支持评论和引用等功能。

很明显，我懒得写编辑器，也懒得找现成的，虽然移植个 web 编辑器不复杂，但是懒癌不可避免。因此，编辑主题是直接用 html 的，如果你不会 html 不意味着你不能发出新的主题，好比你不会 js 无法在弹幕视频网发出高级弹幕，直接打字是可以的，用 html 标签修饰的话也是可以的，当然，js 代码也可以，不要太过分了，整崩溃了就很尴尬了。

权限码并不等价与密码，是一种弱安全策略，只是计算了简单的 hash 数，不过你可以简单的认为这就是密码。如果不输入权限码，那么就没有权限，所有人都能看到你发出的内容。如果你设置了权限码，那么想要看你发送的内容就必须输入权限码，简单的说，就是设了个密码 desu...

补充一下，如果你看代码，就会知道，额...就是说，如果你发出一个新的主题，会截取前 256 个字节，然后从第一个字符到第一个换行为止的内容作为标题，比如编辑了如下内容：

hello world
这是一个 demo
<em>我是斜体字</em>

提交后，“hello world” 就会被当成是标题，如果浏览你发送的主题，html 标签就会有效，显示成下面这样：

hello world 这是一个 demo 我是斜体字

细心的你一定发现了问题的，对，它并没有换行，所以啊，如果你懂 html 呐就不用解释了，不懂的话，告诉你 </br> 就是换行，所以，如果这样编辑：

hello world
</br>这是一个 demo</br>
<em>我是斜体字</em>

就有下面的输出了：

hello world
这是一个 demo
我是斜体字

最后，细心的你还发现，我为什么没有在第一行的末尾使用 </br>，而是把它放在了下一行呢，那是因为在标题里使用 html 标签是不礼貌的，为什么不礼貌，因为我规定的(傲娇脸)
